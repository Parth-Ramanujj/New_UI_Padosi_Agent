How Banner Slides Work
  • Each slide has a Title, Subtitle, a Call-to-Action button, and a Background CSS Class.
  • The Background CSS Class maps to a CSS class in your frontend stylesheet (e.g., banner-new-policy).
  • Hidden slides are stored but not shown on the homepage.
  • Changes take effect immediately after saving.
