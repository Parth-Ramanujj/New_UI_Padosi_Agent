How Push Notifications Work
PWA Required
Agent must have the PadosiAgent PWA installed with notifications allowed.
Instant Delivery
Notifications are sent instantly via Firebase Cloud Messaging and appear as system notifications.
Auto Lead Alerts
New leads from the website automatically send push alerts to the relevant agent — no manual action needed.
No Token = No Push
If an agent hasn't enabled notifications, they won't appear in push lists but will still receive emails.
